# Student Data Organizer

## Project Description
Student Data Organizer is a Python console-based application used to manage student records. It allows users to add, view, update, and delete student information. The application also displays all unique subjects offered by students.

## Features
- Add Student
- Display All Students
- Update Student Information
- Delete Student
- Display Unique Subjects Offered
- Exit Program

## Technologies Used
- Python 3
- Lists
- Dictionaries
- Sets

## Student Information Stored
- Student ID
- Name
- Age
- Grade
- Date of Birth
- Subjects

## How to Run
1. Install Python 3.
2. Save the program as `student_data_organizer.py`.
3. Open Terminal or Command Prompt.
4. Run the command:

```bash
python student_data_organizer.py
